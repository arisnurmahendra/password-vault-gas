# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0.3] - 2026-06-26
### Improved
- **Favicon Fetch:** Menambahkan mekanisme loading state, timeout 10 detik, dan hasil title/favicon yang lebih baik saat mengambil metadata dari URL.
- **Vault Table UI:** Menyempurnakan tampilan tabel agar lebih responsif pada desktop dan mobile, termasuk layout yang lebih rapi dan dukungan tampilan mobile-friendly.

## [1.0.0.2] - 2026-06-17
### Fixed
- **Encryption/Decryption:** Resolved inconsistencies in the encryption-decryption cycle between client-side and server-side relay.

## [1.0.0.1] - 2026-06-12
### Fixed
- **Logic:** Fixed duplicate detection issues when adding new entries with similar site names.

## [1.0.0.0] - 2026-06-12
### Added
- **Initial Release:** Core vault functionality including double-layer encryption, HIBP breach check, and audit logs.
